## Как узнать что не хватает памяти?

### Симптомы:
```
1. Контейнеры часто рестартуются
   docker ps | grep "Restarting"

2. OOM (Out of Memory) ошибки в логах
   docker logs | grep -i OOM

3. Swap активен (очень плохо)
   free -h | grep Swap

4. Приложение становится очень медленным
   curl -w "%{time_total}\n" https://arenda.kapitan-trips.ru
   Время > 2-3 сек = проблема

5. `docker stats` показывает 100% использования памяти
   docker stats --no-stream
```

### Что делать:
```
1. Сначала оптимизировать код/БД
   - Добавить индексы в PostgreSQL
   - Кэшировать в Redis
   - Оптимизировать N+1 queries

2. Если всё оптимизировано, то обновить VPS
   4 CPU + 8 GB → 8 CPU + 16 GB
```
