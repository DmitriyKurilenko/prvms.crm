# Статистика по сделкам

## Для чего эта вкладка

Здесь показаны цифры по работе с сделками:

- Сколько сделок на каждом этапе и на какую сумму (конверсия по воронке).
- Сколько сделок ведёт каждый менеджер и на какую сумму.

## Где её открыть

В левом меню **«Встроенный CRM»** → вверху вкладка **«Статистика»**.

## Пошагово

### Посмотреть статистику по воронке
1. Откройте вкладку **«Статистика»**.
2. Сверху в поле **«Воронка»** выберите нужную.
3. Появятся два блока:
   - **«Конверсия по воронке»** — таблица с этапами, количеством сделок, суммой и долей (полоска «Доля» показывает, какой процент сделок находится на этом этапе).
   - **«Сделки по менеджерам»** — таблица с именами менеджеров, количеством сделок и общей суммой.

### Понять, где «застревают» сделки
1. В блоке **«Конверсия по воронке»** посмотрите, на каком этапе самое большое количество сделок.
2. Если на одном этапе доля сильно больше остальных — это возможно «узкое место».

### Увидеть нагрузку менеджеров
1. Откройте блок **«Сделки по менеджерам»**.
2. Сравните количество сделок и суммы у разных менеджеров.

## Что значат столбцы

### Конверсия по воронке
- **Этап** — название этапа.
- **Сделок** — сколько сделок сейчас на этом этапе.
- **Сумма** — общая сумма сделок на этапе (в рублях).
- **Доля** — полоска, показывающая процент от всех сделок в воронке.

### Сделки по менеджерам
- **Менеджер** — имя сотрудника (если не указан — «Не назначен»).
- **Сделок** — сколько сделок закреплено за менеджером.
- **Сумма** — их общая сумма.

## Типичные ситуации

- **Нужна статистика только по одной воронке** — выберите её в фильтре сверху.
- **Пусто и ничего не видно** — сначала выберите воронку.
- **«Не назначен»** — так подписаны сделки без ответственного. Назначьте менеджера внутри сделки.

## Если не получается

- **Нет вкладки «Статистика»** — у вашей роли нет прав на просмотр сделок. Обратитесь к администратору.
- **Показаны пустые таблицы** — в выбранной воронке пока нет сделок. Попробуйте другую воронку.
- **Цифры кажутся неправильными** — обновите страницу. Статистика может слегка отставать — это нормально.
