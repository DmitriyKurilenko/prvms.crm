# Документы и шаблоны

## Для чего эта страница

Здесь можно:

- Создавать и редактировать **шаблоны** документов (с автоматической подстановкой данных из сделки).
- **Генерировать документы** из шаблонов.
- Смотреть и скачивать уже созданные документы.
- Отправлять документ клиенту на подпись (см. [отдельную статью](08-signing.md)).

Поддерживаемые типы документов:

- **Договор**
- **Акт**
- **Счёт**
- **Оферта**
- **Дополнительное соглашение**
- **Прочее**

При создании организации в разделе сразу появляются **типовые (системные) шаблоны** для каждого вида документа. Их можно использовать как есть, отредактировать или создать собственные.

## Где её открыть

В левом меню нажмите **«Документы»** (значок листа).

Сверху — две вкладки:

- **Документы** — список всех созданных документов.
- **Шаблоны** — список шаблонов, из которых генерируются документы.

## Вкладка «Шаблоны»

### Что такое шаблон

Шаблон — это текст документа, в который можно вставить «поля» (например, имя клиента, сумма, ИНН компании). При генерации поля автоматически заменяются реальными данными сделки.

### Создать новый шаблон

1. Откройте вкладку **«Шаблоны»**.
2. Нажмите **«Новый шаблон»**.
3. В поле **«Название»** напишите, о чём шаблон (например, «Договор поставки»).
4. В поле **«Тип документа»** выберите, к какому виду относится шаблон. От этого зависит тип создаваемого документа.
5. В окне ниже используйте инструменты редактора:
   - **Жирный**, **Курсив**, **Подчёркнутый** — выделение текста.
   - **H1**, **H2**, **P** — заголовок большой, поменьше, обычный абзац.
   - **Список** — маркированный список.
   - **Таблица** — вставить таблицу.
 6. Чтобы вставить **поле из сделки** — нажмите на выпадающий список **«Вставить поле сделки»** и выберите поле (ID сделки, Название сделки, Сумма, Имя контакта, ИНН компании и т.п.). В тексте появится синий значок-метка.
 7. Поставьте галочку **«Активен»**, чтобы шаблон можно было использовать.
 8. Нажмите **«Сохранить»**.

### Изменить существующий шаблон

1. В списке шаблонов нажмите значок карандаша **✏️**.
2. Внесите изменения.
3. Нажмите **«Сохранить»**.

### Посмотреть, как шаблон будет выглядеть

1. В списке нажмите значок «глаз» **👁️** рядом с шаблоном.
2. Откроется предпросмотр.

### Встроенные шаблоны

Рядом с названием шаблона может быть метка **«Встроенный»** — это системные шаблоны. Их можно использовать, но перед изменениями лучше сделать копию.

## Вкладка «Документы»

### Создать документ (сгенерировать)

1. Откройте вкладку **«Документы»**.
2. Нажмите **«Сгенерировать документ»**.
3. Выберите **тип документа** — список шаблонов отфильтруется автоматически.
4. В поле **«Шаблон»** выберите подходящий шаблон (в списке показаны только активные).
5. Необязательно — введите **«ID сделки»**, если хотите привязать документ к конкретной сделке. Номер сделки можно найти в её карточке.
6. Нажмите **«Сгенерировать»**.
7. Документ появится в таблице с нужным типом. Если поля были заполнены — они уже подставлены в текст.

### Создать документ прямо из сделки

1. Откройте сделку в разделе CRM.
2. Внутри сделки нажмите **«Документ»**.
3. Выберите шаблон и нажмите **«Сгенерировать»**.
4. Документ появится в разделе **«Документы»** сделки.

### Посмотреть документ

- Значок **«глаз»** 👁️ — открыть предпросмотр.
- Значок **«скачать»** ⬇️ — скачать PDF.
- Значок **«отправить»** 📤 — отправить клиенту на подпись (см. [статью о подписании](08-signing.md)).

### Статусы документов

- **Черновик** — только что создан, ещё не отправлен.
- **Отправлен** — ссылка для подписи отправлена клиенту.
- **Просмотрен** — клиент открыл ссылку.
- **Подписан** — клиент подписал.
- **Истёк** — срок действия ссылки истёк.

## Типичные ситуации

- **Нужен договор для новой сделки** — создайте из сделки. Там автоматически подхватываются поля клиента и сделки.
- **Менеджер хочет изменить текст документа «под клиента»** — сделайте копию нужного шаблона и отредактируйте её.
- **Нужна таблица в документе** — воспользуйтесь кнопкой **«Таблица»** в редакторе.
- **Хочу видеть только ожидающие подписи** — отсортируйте список по столбцу **«Статус»**.

## Если не получается

- **Кнопка «Сгенерировать» не нажимается** — выберите шаблон в поле **«Шаблон»**.
- **Шаблон не появляется в списке** — проверьте, что у него стоит галочка **«Активен»**.
- **Поля в готовом документе подставились не все** — значит, в сделке эти данные не заполнены (нет контакта, нет ИНН компании и т.п.). Дозаполните сделку и создайте документ заново.
- **Скачивается пустой PDF** — обновите страницу. Если не помогло, попросите коллегу-администратора проверить.
