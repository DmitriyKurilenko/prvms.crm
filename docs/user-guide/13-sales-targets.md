# Планы продаж

## Для чего эта страница

Здесь вы ставите менеджерам планы на месяц — по сумме и по количеству сделок — и следите за их выполнением (факт против плана).

## Где её открыть

В левом меню нажмите **«Планы продаж»** (значок флажка). Раздел доступен владельцу и администратору.

## Пошагово

### Задать план менеджеру
1. Выберите **«Месяц»**.
2. Выберите **«Менеджера»**.
3. Укажите план: **сумму** и **«План по сделкам»** (количество).
4. Нажмите **«Сохранить план»**.

Если план на этот месяц уже был задан, он обновится.

### Посмотреть выполнение
Блок **«Выполнение за <месяц>»** показывает по каждому менеджеру:
- **Менеджер**.
- **Сумма (факт / план)** — сколько заработано против плана.
- **Сделки (факт / план)** — сколько сделок закрыто против плана.

## Типичные ситуации

- **Поставить всем отделам цель на месяц** — задайте план каждому менеджеру по очереди.
- **Скорректировать план в середине месяца** — выберите тот же месяц и менеджера, введите новые значения и сохраните.

## Если не получается

- **Нет раздела «Планы продаж»** — он доступен только владельцу и администратору.
- **Факт не растёт** — факт считается по закрытым (выигранным) сделкам выбранного месяца. Проверьте, что сделки переведены в этап «Выигрыш».
- **Менеджера нет в списке** — добавьте сотрудника с ролью «Менеджер» в разделе [Команда](22-team.md).
