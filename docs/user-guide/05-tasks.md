# Задачи

## Для чего эта вкладка

Список задач по сделкам: что сделать, к какому сроку и по какой сделке. Задачи могут появляться автоматически (через триггеры воронки) или создаваться вручную из заметок в сделке.

## Где её открыть

В левом меню **«Встроенный CRM»** → вверху вкладка **«Задачи»**.

## Что показывает таблица

- **Задача** — название. Если задача привязана к сделке, название — это ссылка, по клику открывается сделка.
- **Статус** — «Запланирована», «Выполнена», «Отменена» и т.п.
- **Срок** — до какой даты нужно сделать.
- **Создана** — когда появилась задача.

Сверху — фильтр **«Все статусы»**: можно показать только выполненные, только запланированные и так далее.

## Пошагово

### Отфильтровать задачи
1. Сверху над таблицей нажмите на выпадающий список **«Все статусы»**.
2. Выберите нужный статус.
3. Список обновится автоматически.

### Отметить задачу как выполненную
1. Найдите задачу со статусом «Запланирована».
2. Справа в строке нажмите галочку **«✓»** (подсказка — «Выполнено»).
3. Статус изменится на «Выполнена».

### Удалить задачу
1. Справа в строке задачи нажмите корзину **🗑️**.
2. Подтвердите действие.

### Перейти в сделку задачи
1. Нажмите на название задачи, если оно показано как ссылка.
2. Откроется окно сделки, к которой привязана задача.

## Типичные ситуации

- **Нужно увидеть просроченные задачи** — отсортируйте по столбцу **«Срок»**: нажмите на заголовок столбца.
- **Много задач накопилось** — переключите фильтр на «Запланирована», чтобы увидеть только актуальные.
- **Задачи появляются сами** — это работают триггеры воронки. Их настройка описана в статье [Воронки и триггеры](04-pipelines-and-triggers.md).
- **Добавить новую задачу вручную** — откройте нужную сделку, в разделе «Лог активности» выберите тип **«✅ Задача»**, напишите, что сделать, и нажмите **«Добавить»**.

## Если не получается

- **Нет вкладки «Задачи»** — вкладка доступна ролям с доступом к CRM (обычно менеджерам и выше). Проверьте свою роль.
- **Не появляется кнопка «Выполнено»** — скорее всего, задача уже не «Запланирована». Галочка появляется только у запланированных задач.
- **Список пустой** — либо задач действительно нет, либо выбранный фильтр их скрывает. Сбросьте фильтр.
