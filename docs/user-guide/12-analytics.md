# Аналитика CRM

## Для чего эта страница

Здесь собраны цифры по продажам: насколько хорошо сделки проходят воронку, какая конверсия, прогноз закрытия, причины потерь и нагрузка менеджеров.

## Где её открыть

В левом меню нажмите **«Аналитика»** (значок графика). Сверху можно выбрать **воронку** и период.

## Что показывает страница

### Воронка и конверсия
Сводка по сделкам:
- **Всего сделок**.
- **Выиграно**.
- **Проиграно**.
- **В работе**.
- **Конверсия** — доля выигранных сделок.

### Прогноз закрытия
- **Открытый пайплайн** — общая сумма сделок в работе.
- **Сделок в работе** — их количество.
- **Прогноз за период** — ожидаемая сумма к закрытию.
- **Сделок к закрытию** — сколько сделок планируется закрыть.

### Причины потерь
Таблица: **Причина**, количество сделок и сумма по каждой причине проигрыша. Помогает понять, почему срываются сделки.

### Конверсия по воронке
Таблица по этапам: **Этап**, **Сделок**, **Сумма** и **Доля** (полоска показывает процент сделок на этапе). По ней видно, где сделки «застревают».

### Сделки по менеджерам
Таблица: **Менеджер**, **Сделок**, **Сумма**. Показывает нагрузку и результаты сотрудников.

## Пошагово

### Посмотреть аналитику по воронке
1. Откройте **«Аналитика»**.
2. Сверху выберите нужную **воронку** и период.
3. Изучите блоки: конверсия, прогноз, причины потерь, этапы, менеджеры.

### Найти «узкое место»
В блоке **«Конверсия по воронке»** найдите этап с самой большой долей сделок — там обычно скапливаются «зависшие» сделки.

## Типичные ситуации

- **Нужен прогноз выручки на месяц** — выберите период и смотрите блок «Прогноз закрытия».
- **Понять, почему теряем клиентов** — изучите «Причины потерь» (заполняйте «Причину проигрыша» в сделках, иначе блок будет пустым).
- **Сравнить менеджеров** — блок «Сделки по менеджерам».

## Если не получается

- **Нет раздела «Аналитика»** — у роли нет прав на просмотр сделок. Обратитесь к администратору.
- **Пустые таблицы** — в выбранной воронке нет сделок либо не заполнены причины проигрыша.
- **Цифры кажутся неточными** — обновите страницу; аналитика может слегка отставать.
