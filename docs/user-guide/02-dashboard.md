# Дашборд

## Для чего эта страница

Дашборд — это первая страница, которую вы видите после входа. На ней собраны ключевые цифры по работе организации (KPI), список разделов и текущее использование лимитов тарифа.

## Где её открыть

В левом меню нажмите **«Дашборд»** (значок домика). Это самый первый пункт меню.

## Что есть на странице

### Блоки с цифрами (KPI)

Сверху — четыре показателя:

- **Менеджеры онлайн** — сколько сотрудников сейчас работает в системе. Число обновляется автоматически.
- **Открытых сделок** — сколько сделок сейчас в работе (не выиграны и не проиграны).
- **Документы (месяц)** — сколько документов создано за текущий месяц.
- **Распределено заявок** — сколько заявок автоматически назначено менеджерам.

### Блок «Разделы»

Плитки-ссылки для быстрого перехода в основные разделы: Контакты, Сделки, Задачи, Документы, Телефония, Мессенджеры, Интеграции, Распределение. Плитка раздела, который недоступен в вашем тарифе или для вашей роли, будет неактивной.

### Блок «Использование»

Показывает, насколько вы приблизились к лимитам тарифа:

- **Документы в этом месяце** — сколько документов уже создано из месячного лимита.
- **Менеджеры** — сколько менеджеров заведено из доступного количества.
- **Пайплайны (воронки)** — сколько воронок создано из лимита.

## Пошагово

### Посмотреть цифры
1. Откройте **«Дашборд»** — показатели обновляются автоматически. Счётчик «Менеджеры онлайн» подтягивается в реальном времени.

### Быстро перейти в раздел
1. В блоке **«Разделы»** нажмите нужную плитку.
2. Откроется соответствующий раздел.

## Типичные ситуации

- **Цифры показывают 0** — скорее всего, в системе ещё нет данных (нет менеджеров, нет сделок). Это нормально для нового аккаунта.
- **Плитка раздела неактивна** — раздел недоступен по вашему тарифу или роли. Проверьте раздел [«Подписка»](29-subscription.md).
- **Подходите к лимиту** — в блоке «Использование» полоска заполнена почти полностью. Подумайте о смене тарифа.

## Если не получается

- Если страница не открывается — обновите её (клавиша **F5**).
- Если плитки быстрого перехода не срабатывают — возможно, раздел недоступен по вашей роли или тарифу.
