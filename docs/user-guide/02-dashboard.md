# Главная

## Для чего эта страница

«Главная» — это первая страница, которую вы видите после входа. На ней собраны основные цифры по вашей работе и быстрые кнопки для частых действий: создать сделку, создать договор, синхронизировать менеджеров.

## Где её открыть

В левом меню нажмите **«Главная»** (значок домика).

## Что есть на странице

Сверху — четыре блока с цифрами (KPI):

- **Менеджеры** — сколько менеджеров заведено в вашей организации.
- **Договоры** — сколько договоров создано.
- **Распределено заявок** — сколько заявок сегодня автоматически распределено между менеджерами.
- **Открытых сделок** — сколько сделок сейчас в работе.

Ниже — блок **«Быстрые действия»** с кнопками:

- **Создать сделку** — откроет раздел CRM для новой сделки.
- **Создать договор** — перейдёт в раздел договоров.
- **Синхронизировать менеджеров** — откроет раздел интеграций, где можно обновить список менеджеров из внешней CRM.

## Пошагово

### Посмотреть цифры
1. Откройте **«Главная»** — цифры обновляются автоматически.

### Быстро создать сделку
1. Нажмите **«Создать сделку»**.
2. Вы попадёте в CRM. Дальше действуйте как в разделе [Встроенный CRM](03-crm.md).

### Быстро создать документ
1. Нажмите **«Документ»**.
2. Вы окажетесь в разделе **«Документы»**. Подробнее — в статье [Документы и шаблоны](07-documents.md).

## Типичные ситуации

- **Цифры показывают 0** — скорее всего, в системе ещё нет данных (нет менеджеров, нет сделок). Это нормально для нового аккаунта.
- **Некоторые цифры не обновляются** — иногда статистика подключена не во всех тарифах. Тогда показывается значение «0».

## Если не получается

- Если страница не открывается — обновите её (клавиша **F5** или кнопка обновления в браузере).
- Если кнопки быстрых действий не срабатывают — проверьте, открыт ли у вас нужный раздел в меню слева. Возможно, он недоступен по вашей роли или тарифу.
