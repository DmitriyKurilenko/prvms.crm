# Чаты: переписка из мессенджеров

## Для чего эта страница

«Чаты» — единое окно переписки с клиентами из всех подключённых мессенджеров (Telegram, WhatsApp, MAX, ВКонтакте) и электронной почты. Вы видите все обращения в одном месте и отвечаете, не переключаясь между приложениями.

## Где её открыть

В левом меню нажмите **«Чаты»** (значок самолётика).

> Подключение самих каналов выполняется отдельно — см. [Мессенджер-каналы](17-channels.md).

## Как устроено окно

- Сверху — выбор **канала** (какой мессенджер показывать).
- Слева — список диалогов (имя клиента или идентификатор, время последнего сообщения).
- Справа — переписка с выбранным клиентом и поле для ответа.

## Пошагово

### Ответить клиенту
1. Сверху выберите нужный **канал**.
2. В списке слева нажмите на диалог.
3. Справа откроется переписка.
4. В поле ввода напишите ответ и нажмите **«Отправить»**.

Ваши сообщения отображаются с одной стороны, сообщения клиента — с другой. Если у диалога есть связанная сделка, она будет отмечена.

### Передать диалог AI-ассистенту
В окне чата есть кнопка перехода к [AI-ассистенту](25-assistant.md) — он поможет составить ответ по контексту переписки.

## Пометки на сообщениях

- Красный значок ошибки — сообщение не удалось доставить.
- Подпись под сообщением — причина ошибки доставки.

## Типичные ситуации

- **Клиент написал впервые** — если в канале включено создание сделки при первом сообщении, сделка уже создана; диалог появится в списке.
- **Нужно видеть переписку из разных мессенджеров** — переключайте канал сверху.
- **Сообщение «зависло» с ошибкой** — проверьте статус канала в [настройках мессенджеров](17-channels.md).

## Если не получается

- **Нет раздела «Чаты»** — мессенджеры подключаются в тарифе. Проверьте раздел [«Подписка»](29-subscription.md).
- **Нет диалогов** — выбран не тот канал либо клиенты ещё не писали.
- **Не отправляется ответ** — проверьте интернет и статус канала; при ошибке у сообщения появится красная пометка.
