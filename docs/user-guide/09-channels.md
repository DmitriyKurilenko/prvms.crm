# Мессенджер-каналы (Telegram, WhatsApp, MAX)

## Для чего эта страница

Подключить мессенджеры к CRM, чтобы:

- Принимать сообщения клиентов прямо в системе.
- Отвечать им из одного окна.
- Автоматически создавать сделки при первом обращении.

## Где её открыть

В левом меню нажмите **«Мессенджеры»** (значок самолётика).

Сверху — две вкладки:

- **Каналы** — список и настройка подключений.
- **Чаты** — переписка с клиентами.

---

## Вкладка «Каналы»

### Что означают поля

- **Название** — как вы назовёте канал у себя (например, «Основной Telegram-бот»).
- **Тип** — какой это мессенджер:
  - Telegram Bot.
  - WhatsApp (провайдер).
  - WhatsApp Business API.
  - MAX.
- **Bot Token** (для Telegram и MAX) — ключ, который выдаёт сам мессенджер при создании бота. Его нужно получить у администратора или в настройках бота.
- **Send URL** и **Auth Token** (для WhatsApp) — адрес и ключ вашего провайдера WhatsApp.
- **Webhook Token (опц.)** — секретный код для проверки подлинности запросов (необязательно).
- **Создавать сделку при первом сообщении** — галочка: если включена, при первом сообщении от нового клиента в CRM автоматически создастся сделка.
- **Приветственное сообщение** — текст, который отправляется клиенту при первом обращении.

### Подключить новый канал

1. Откройте вкладку **«Каналы»**.
2. В верхнем блоке **«Подключить канал»** заполните:
   - **Название** и **Тип**.
   - Поля авторизации (Bot Token или Send URL / Auth Token — зависит от типа).
   - По желанию — приветственное сообщение и галочку «Создавать сделку при первом сообщении».
3. Нажмите **«Подключить»**.
4. Канал появится в списке ниже.
5. Для Telegram и MAX: в строке нового канала нажмите значок **«Зарегистрировать webhook»** 🔗. Это скажет мессенджеру присылать сообщения в систему.

### Редактировать канал

1. В списке нажмите значок карандаша **✏️**.
2. Поменяйте нужные поля.
3. Нажмите **«Сохранить»**.
4. Тип канала изменить после подключения нельзя.

### Отключить или удалить канал

- Значок **выключателя** ⏻ — временно отключить или включить канал.
- Значок **корзины** 🗑️ — удалить.

### Webhook URL

В колонке **«Webhook URL»** показан адрес вашего канала. Нажмите на него — адрес скопируется. Этот адрес иногда нужно вставить в настройки самого мессенджера (у провайдера WhatsApp).

### Статусы канала

- **Активен** — канал работает, сообщения приходят.
- **Ошибка** — что-то не так (проверьте токен, интернет, Webhook).
- **Отключён** — канал выключен вручную.

## Вкладка «Чаты»

Здесь ведётся переписка с клиентами.

### Структура окна

Слева — список чатов и фильтр по каналу. Справа — окно сообщений.

### Пошагово

1. Откройте вкладку **«Чаты»**.
2. В поле **«Канал»** выберите нужный мессенджер.
3. В левой колонке появится список чатов (имя клиента или идентификатор, дата последнего сообщения).
4. Нажмите на чат, чтобы увидеть переписку справа.
5. В поле **«Написать ответ…»** введите текст и нажмите значок **«Отправить»** ✈️.
6. Ваши сообщения показываются справа, клиентские — слева.
7. Если у чата есть сделка — внизу будет метка **«📋 Сделка #…»**.

### Пометки на сообщениях

- **✕** красным — сообщение не удалось доставить.
- Красная подпись под сообщением — причина ошибки.

---

## Типичные ситуации

- **Клиент написал впервые — где взять сделку?** — если в канале включена галочка **«Создавать сделку при первом сообщении»**, она появилась автоматически. Откройте вкладку **«Чаты»** — рядом с именем клиента будет ссылка на сделку.
- **Нужно отключить канал на выходные** — нажмите значок выключателя **⏻** в строке канала.
- **Пришёл новый провайдер WhatsApp** — создайте новый канал, введите новые Send URL и Auth Token.
- **Нужно видеть переписку из разных каналов** — переключайте в выпадающем «Канал».

## Если не получается

- **Нет раздела «Мессенджеры»** — он подключается в тарифе. Проверьте в разделе **«Подписка»**.
- **Статус «Ошибка» у канала** — наведите на значок **ⓘ** рядом со статусом: там будет подсказка. Чаще всего причина — неверный Bot Token или не зарегистрирован webhook (нажмите 🔗).
- **Сообщения не приходят** — проверьте, активен ли канал, правильный ли токен. Для Telegram обязательно нажать **«Зарегистрировать webhook»**.
- **Не могу отправить сообщение** — проверьте интернет, статус канала. Если рядом с вашим сообщением **✕** — канал настроен неправильно.
- **Нет чатов в списке** — либо выбран не тот канал, либо клиенты ещё не писали.
