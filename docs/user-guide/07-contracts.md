# Договоры и шаблоны

## Для чего эта страница

Здесь можно:

- Создавать и редактировать **шаблоны** договоров (с автоматической подстановкой данных из сделки).
- **Генерировать договоры** из шаблонов.
- Смотреть и скачивать уже созданные договоры.
- Отправлять договор клиенту на подпись (см. [отдельную статью](08-signing.md)).

## Где её открыть

В левом меню нажмите **«Договоры»** (значок листа).

Сверху — две вкладки:

- **Договоры** — список всех созданных договоров.
- **Шаблоны** — список шаблонов, из которых генерируются договоры.

---

## Вкладка «Шаблоны»

### Что такое шаблон

Шаблон — это текст договора, в который можно вставить «поля» (например, имя клиента, сумма, ИНН компании). При генерации поля автоматически заменяются реальными данными сделки.

### Создать новый шаблон

1. Откройте вкладку **«Шаблоны»**.
2. Нажмите **«Новый шаблон»**.
3. В поле **«Название»** напишите, о чём шаблон (например, «Договор поставки»).
4. В окне ниже используйте инструменты редактора:
   - **Жирный**, **Курсив**, **Подчёркнутый** — выделение текста.
   - **H1**, **H2**, **P** — заголовок большой, поменьше, обычный абзац.
   - **Список** — маркированный список.
   - **Таблица** — вставить таблицу.
5. Чтобы вставить **поле из сделки** — нажмите на выпадающий список **«Вставить поле сделки»** и выберите поле (ID сделки, Название сделки, Сумма, Имя контакта, ИНН компании и т.п.). В тексте появится синий значок-метка.
6. Поставьте галочку **«Активен»**, чтобы шаблон можно было использовать.
7. Нажмите **«Сохранить»**.

### Изменить существующий шаблон

1. В списке шаблонов нажмите значок карандаша **✏️**.
2. Внесите изменения.
3. Нажмите **«Сохранить»**.

### Посмотреть, как шаблон будет выглядеть

1. В списке нажмите значок «глаз» **👁️** рядом с шаблоном.
2. Откроется предпросмотр.

### Встроенные шаблоны

Рядом с названием шаблона может быть метка **«Встроенный»** — это системные шаблоны. Их можно использовать, но перед изменениями лучше сделать копию.

## Вкладка «Договоры»

### Создать договор (сгенерировать)

1. Откройте вкладку **«Договоры»**.
2. Нажмите **«Сгенерировать договор»**.
3. В поле **«Шаблон»** выберите подходящий шаблон (в списке показаны только активные).
4. Необязательно — введите **«ID сделки»**, если хотите привязать договор к конкретной сделке. Номер сделки можно найти в её карточке.
5. Нажмите **«Сгенерировать»**.
6. Договор появится в таблице. Если поля были заполнены — они уже подставлены в текст.

### Создать договор прямо из сделки

1. Откройте сделку в разделе CRM.
2. Внутри сделки нажмите **«Договор»**.
3. Выберите шаблон и нажмите **«Сгенерировать»**.
4. Договор появится в разделе **«Договоры»** сделки.

### Посмотреть договор

- Значок **«глаз»** 👁️ — открыть предпросмотр.
- Значок **«скачать»** ⬇️ — скачать PDF.
- Значок **«отправить»** 📤 — отправить клиенту на подпись (см. [статью о подписании](08-signing.md)).

### Статусы договоров

- **Черновик** — только что создан, ещё не отправлен.
- **Отправлен** — ссылка для подписи отправлена клиенту.
- **Просмотрен** — клиент открыл ссылку.
- **Подписан** — клиент подписал.
- **Истёк** — срок действия ссылки истёк.

## Типичные ситуации

- **Нужен договор для новой сделки** — создайте из сделки. Там автоматически подхватываются поля клиента и сделки.
- **Менеджер хочет изменить текст договора «под клиента»** — сделайте копию нужного шаблона и отредактируйте её.
- **Нужна таблица в договоре** — воспользуйтесь кнопкой **«Таблица»** в редакторе.
- **Хочу видеть только ожидающие подписи** — отсортируйте список по столбцу **«Статус»**.

## Если не получается

- **Кнопка «Сгенерировать» не нажимается** — выберите шаблон в поле **«Шаблон»**.
- **Шаблон не появляется в списке** — проверьте, что у него стоит галочка **«Активен»**.
- **Поля в готовом договоре подставились не все** — значит, в сделке эти данные не заполнены (нет контакта, нет ИНН компании и т.п.). Дозаполните сделку и создайте договор заново.
- **Скачивается пустой PDF** — обновите страницу. Если не помогло, попросите коллегу-администратора проверить.
