# Интеграции с внешними CRM

## Для чего эта страница

Связать систему с уже используемой CRM (amoCRM или Битрикс24), чтобы данные передавались между ними автоматически: контакты, сделки, менеджеры.

## Где её открыть

В левом меню нажмите **«Интеграции»** (значок розетки). Раздел доступен владельцу и администратору.

> Пункт «Интеграции» может быть неактивным (с замком), если интеграции отключены в вашей конфигурации или тарифе. В этом случае обратитесь к владельцу аккаунта.

## Два режима подключения

Сверху — переключатель:

- **Рекомендуемый: приложение** — установка через приложение в маркетплейсе amoCRM/Битрикс24: нажимаете кнопку, подтверждаете доступ — всё готово.
- **Быстрый старт: webhook** — если у вас уже есть ссылка-webhook от внешней CRM.

Первый вариант надёжнее.

## Вариант «Приложение»

1. Выберите **«Рекомендуемый: приложение»**.
2. Впишите **«Название подключения»**.
3. Нажмите **«Установить amoCRM приложение»** или **«Установить Битрикс24 приложение»**.
4. На странице вашей CRM подтвердите разрешения.
5. Вас вернёт обратно — появится строка подключения.

## Вариант «Webhook»

1. Выберите **«Быстрый старт: webhook»**, затем CRM и название.
2. Для **amoCRM**: Base URL, Access Token, при необходимости HMAC-секрет.
3. Для **Битрикс24**: Webhook URL и при необходимости дополнительные параметры авторизации.
4. Нажмите **«Создать webhook-подключение»**.

## Таблица подключений

Показывает: **Название**, **CRM**, **Режим**, **Статус** (зелёный — работает, жёлтый — нужна доавторизация или не хватает прав, красный — ошибка), **Scope**, **Webhook URL** и время последней активности.

Действия в строке: **«Проверить»**, **«Синхронизировать менеджеров»**, **«Переавторизовать»**, **«Журнал ошибок»** (с числом ошибок), **«Удалить»**.

## Журнал ошибок

Показывает метку важности (INFO/WARNING/ERROR), описание и строку **«Что сделать»** с подсказкой по исправлению.

## Типичные ситуации

- **Первое подключение amoCRM** — режим «Приложение», название, «Установить amoCRM приложение», подтвердить доступ.
- **Интеграция отвалилась** — нажмите **«Переавторизовать»**.
- **Новые менеджеры не подтянулись** — нажмите **«Синхронизировать менеджеров»**.

## Если не получается

- **Раздел недоступен или с замком** — доступ только у владельца/администратора и при включённых интеграциях.
- **Жёлтый статус «Не хватает прав»** — откройте журнал ошибок, нажмите «Переавторизовать» и разрешите всё.
- **Webhook не приходит** — скопируйте адрес из столбца «Webhook URL» и вставьте его в настройки внешней CRM.
