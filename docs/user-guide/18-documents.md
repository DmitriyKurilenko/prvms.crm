# Документы и шаблоны

## Для чего эта страница

Здесь можно:

- Создавать и редактировать **шаблоны** документов (с автоматической подстановкой данных из сделки).
- **Генерировать документы** из шаблонов.
- Смотреть и скачивать уже созданные документы.
- Отправлять документ клиенту на подпись (см. [отдельную статью](19-signing.md)).

Поддерживаемые типы документов: **Договор**, **Акт**, **Счёт**, **Оферта**, **Дополнительное соглашение**, **Прочее**.

При создании организации сразу появляются **типовые (системные) шаблоны** для каждого вида документа. Их можно использовать как есть, отредактировать или создать собственные.

## Где её открыть

В левом меню нажмите **«Документы»** (значок листа).

Сверху — две вкладки: **«Документы»** (список созданных) и **«Шаблоны»** (заготовки).

---

## Вкладка «Шаблоны»

### Что такое шаблон
Шаблон — это текст документа с «полями» (имя клиента, сумма, ИНН компании). При генерации поля заменяются реальными данными сделки, включая список позиций товаров.

### Создать шаблон
1. Откройте вкладку **«Шаблоны»** и нажмите **«Новый шаблон»**.
2. Впишите **«Название»**.
3. Выберите **«Тип документа»** — от него зависит тип создаваемого документа.
4. Оформите текст в редакторе (жирный, курсив, заголовки, списки, таблицы).
5. Вставьте поля сделки через список **«Вставить поле сделки»** (ID сделки, сумма, имя контакта, ИНН компании и т.п.).
6. Поставьте галочку **«Активен»**, чтобы шаблон можно было использовать.
7. Нажмите **«Сохранить»**.

### Изменить или посмотреть шаблон
- Карандаш ✏️ — редактировать.
- Глаз 👁️ — предпросмотр.

Метка **«Встроенный»** означает системный шаблон — перед изменением лучше сделать копию.

## Вкладка «Документы»

### Сгенерировать документ
1. Откройте вкладку **«Документы»** и нажмите **«Сгенерировать документ»**.
2. Выберите **тип документа** — список шаблонов отфильтруется.
3. Выберите **«Шаблон»** (показаны только активные).
4. По желанию укажите сделку, чтобы привязать документ и подставить её данные.
5. Нажмите **«Сгенерировать»**.

### Создать документ из сделки
Откройте сделку и в блоке «Документы» сгенерируйте документ — данные клиента и сделки подставятся автоматически.

### Действия над документом
- Глаз 👁️ — предпросмотр.
- Скачать ⬇️ — скачать PDF.
- Отправить 📤 — отправить клиенту на подпись (см. [Подписание](19-signing.md)).

### Статусы документов
**Черновик** → **Отправлен** → **Просмотрен** → **Подписан**. Возможен статус **Истёк**, если срок ссылки вышел.

## Типичные ситуации

- **Нужен договор для сделки** — генерируйте из сделки, поля подставятся сами.
- **Текст «под клиента»** — сделайте копию шаблона и отредактируйте.
- **Счёт с позициями товаров** — добавьте позиции в сделку, в шаблоне используйте поле со списком позиций.

## Если не получается

- **Кнопка «Сгенерировать» неактивна** — не выбран шаблон.
- **Шаблон не в списке** — у него нет галочки «Активен».
- **Поля подставились не все** — в сделке не заполнены эти данные. Дозаполните и сгенерируйте заново.
- **Нет раздела «Документы»** — модуль документооборота подключается в тарифе.
